Basic Tracer¶

This example shows how to use OpenTelemetry to instrument a Python application - e.g. a batch job. It supports exporting spans either to the console or to Jaeger.

The source files required to run this example are available here.

Run the application¶

Console¶

  • Run the sample

$ python tracer.py

The output will be displayed at the console

Jaeger¶

Setup Jaeger Tracing.

  • Run the sample

$ pip install opentelemetry-ext-jaeger
$ EXPORTER=jaeger python tracer.py

The traces should be available in the Jaeger UI at http://localhost:16686

Collector¶

  • Start Collector

$ pip install docker-compose
$ cd docker
$ docker-compose up

  • Run the sample

$ pip install opentelemetry-ext-otcollector
$ EXPORTER=collector python tracer.py

Collector is configured to export to Jaeger, follow Jaeger UI instructions to find the traces.
